# Fun Kanban V2.1 Screenshots

Screenshots for GitHub repository showcase.

## Screenshots

### 01-login-desktop.png
Desktop login screen (1920x1080) showing the Fun Kanban V2.1 authentication interface with dark theme.

### 02-workspace-desktop.png
Desktop kanban board view (1920x1080) displaying the full workspace with tasks organized in To Do, In Progress, and Done columns. Features include:
- Task cards with priority indicators (colored dots)
- Due dates and last updated timestamps
- Drag-and-drop task management
- Project filtering and sorting options
- Dark mode theme

### 03-workspace-mobile.png
Mobile kanban board view (375x667) with responsive design featuring:
- Vertical task list optimized for mobile
- Bottom navigation tabs (To Do, Progress, Done, Archive)
- Touch-friendly task cards
- Clean, modern mobile UI

### 04-dashboard-desktop.png
Desktop dashboard view (3014x1854) showing the main workspace selection screen with:
- Sidebar navigation (Dashboard, Workspaces, Admin Panel)
- Workspace cards with task count and member info
- "New Workspace" creation button
- User profile display with admin@demo.com
- Dark theme interface
